VirtualBoxを用いたWindows(Mac)のコンピュータでの解析(設定法)

[日本語トップページへのリンク]

Linuxの走るコンピュータを、すぐには準備できないかもしれません。
VirtualBoxは、現在オラクルが提供している仮想化ソフトウェアで、WindowsやマッキントッシュのOS上でLinuxを動かすことができます。

  • 図をクリックすると拡大されます。
  • Windowsあるいはマッキントッシュ(OS X)の中に仮想のLinuxのデスクトップができます。
  • Windowsが走りながら、その中にゲストOSのアプリケーションとして動くので、動作は遅いです。
  • Linuxの専用マシンを準備する前にPEDの動作の確認がしたい場合、
    専用マシンを準備するほどの配列データはない場合など、時々使うような場合には便利です。

インストール

  • VirtualBoxのダウンロードサイトよりWindowsあるいはOS Xホスト用のファイルをダウンロードします。
  • Ubuntu Desktop 日本語 Remixより日本語版のISOイメージのファイルをダウンロードします。
  • BIOSのVirtualization TechnologyがEnabeledになっている必要があります。
    パソコンをF2やF10(機種によって違います)キーを押しながら再起動して、BIOS画面を表示させて、
    CPUのオプションからVirtualization Technologyを探して、有効(Enabeled)にしてから設定をセーブして、OSを起動します。
    画面はHPのノートブックの例です。
  • インストールしたOracle VM virtualBoxマネージャーを起動して、新規をクリックします。

    名前にUbuntuと入れると、タイプ: Linux、バージョン: Ubuntu(64-bit)が自動的に選択されるようです。違っている場合は、図のように選択してください。
  • メモリサイズは4Gバイトにします。

  • 「仮想ハードディスクを作成する」を選んで作成をクリックします。

  • 仮想ハードディスクのファイルのタイプはVDIを選びます。

  • 仮想ハードディスクのファイルは可変サイズを選びます。

  • サイズはウイルスやバクテリアの解析なら50Gバイトあれば充分です。ヒトなどの大きなゲノムサイズでは数100GBは必要です。

  • 光学ドライブのリンクをクリックして、事前にダウンロードしておいたubuntu-ja-20.04-desktop-amd64.isoを指定します。

  • 起動をクリックすると、isoファイルが読み込まれてインストール画面が表示されます。
    Ubuntuをインストールをクリックして、指示に従い、すべてデフォルトでインストールします。

  • インストールが完了したら、設定したパスワードでログインします。
    左下のアプリケーションを起動するのボタンをクリックして、アプリをすべて表示にして、ページを繰って「端末」をクリックして起動します。

  • 開いた端末で、必要なソフトをダウンロードとインストールを行います。 
    $ sudo apt update
$ sudo apt upgrade
$ sudo apt install curl
$ sudo apt install git
$ git clone https://github.com/akiomiyao/ped.git
  • ここをクリックしてsra-toolkitをインストールと設定を行います。
  • PEDでの解析は、解析例を参照してください。
